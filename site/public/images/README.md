# Bilder / Photos

Drop the real photos from vaskmedmeg.no here and reference them from the
components (e.g. `<img src="/images/hero.jpg" />`), or swap the SVG
`Illustration` in the hero for a real photograph.

Suggested images to add:
- `hero.jpg` — a bright, clean Nordic living room or a cleaner at work (used in the hero panel)
- `service-*.jpg` — one per service card (optional)
- `team.jpg` / `about.jpg` — for a future "Om oss" section

Files placed in `public/` are served from the site root, so
`public/images/hero.jpg` is available at `/images/hero.jpg`.

Keep images optimized (WebP/JPEG, < 300 KB each) for fast loading.
